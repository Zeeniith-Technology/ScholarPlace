## PART 1: HISTORICAL & FOUNDATIONAL CONTEXT

### Why Ratio Was Invented (Historical Context)

**Ancient Problem (3000 BCE):**

```
Babylonian merchants needed to trade:
- 5 measures of grain ‚Üî 2 measures of gold
- But how to trade with strangers?
- How to verify fairness?

Without ratio:
"I'm giving 100 measures of grain for 40 measures of gold"
Is this fair? Unknown!

With ratio:
"This trade maintains 5:2 ratio"
Both can verify fairness by checking same ratio

Why this works:
5 : 2 = 100 : 40
If ratio maintained ‚Üí trade is fair
```

**Why ratio matters today:**
- Fair trade across different quantities
- Scaling recipes, maps, architectural plans
- Understanding relationships (not just numbers)
- Business partnerships (who gets what share)

## PART 2: DEEP DEFINITION OF RATIO

### Three Levels of Understanding

**Level 1 (Formal Mathematical):**
Ratio is an ordered pair of quantities (a, b), often expressed as a:b, representing the relationship a/b.

**Level 2 (Conceptual):**
Ratio is a proportional relationship showing "for every a units of one thing, there are b units of another."

**Level 3 (Practical):**
Ratio is a tool to compare things of different sizes by finding their proportional relationship, which remains constant even when quantities change.

### Why Each Definition Matters

```
Definition 1 (Formal): Helps with mathematical operations
Definition 2 (Conceptual): Helps understand real-world meaning  
Definition 3 (Practical): Helps solve actual problems
```

## PART 3: RATIO TYPES

### Type 1: Part-to-Part Ratio

**Definition:** Comparing two different parts of the same whole

```
Example: In a class of 30 students:
- 18 boys, 12 girls
- Boys:Girls = 18:12 = 3:2

This means: For every 3 boys, there are 2 girls
```

---

### Type 2: Part-to-Whole Ratio

**Definition:** Comparing one part to the entire whole

```
Example: In a class of 30 students with 18 boys:
Boys:Class = 18:30 = 3:5

Meaning:
- Boys are 3/5 of the class
- Boys are (3/5) √ó 100 = 60% of class

Connection to percentages from Week 2!
```

## PART 2: COMPOUND RATIO SOLUTION METHOD

### Step-by-Step Process

**Step 1: Identify the common quantity**
In this case, B appears in both ratios

**Step 2: Find LCM of B values**
B = 4 in first ratio
B = 2 in second ratio
LCM(4, 2) = 4

**Step 3: Scale ratios to make B equal**
First ratio A:B = 3:4 (already has B=4) ‚úì
Second ratio B:C = 2:5 (need B=4, so multiply by 2)
‚Üí B:C = 4:10

**Step 4: Combine all three**
A:B:C = 3:4:10

**Verification:**
- A:B = 3:4 ‚úì
- B:C = 4:10 = 2:5 ‚úì

## PART 1: DIRECT PROPORTION MASTERY

### Definition
If y = kx (where k is constant), then y is directly proportional to x

### How to Identify
- As x increases, y increases
- Ratio y/x always constant
- Graph is straight line through origin

### Real Applications

```
1. Total Cost = Price per unit √ó Quantity
2. Distance = Speed √ó Time (at constant speed)
3. Simple Interest = Principal √ó Rate √ó Time
4. Work done = Rate √ó Time
```

---

### Example 3.1: Direct Proportion - Simple Interest

```

   Principal ‚Çπ5,000, Rate 8% per year
   
   Make table for SI vs Time
   Find constant of proportionality

SI = 400 √ó T

Time (years) | SI (‚Çπ)
1            | 400

4            | 1,600

This is direct proportion: SI ‚àù T

```

## PART 2: INVERSE PROPORTION MASTERY

### Definition
If xy = k (or y = k/x), then y is inversely proportional to x

### How to Identify
- As x increases, y decreases
- Product xy always constant
- Graph is hyperbola curve

### Real Applications

```
1. Speed √ó Time = Distance (constant)
2. Number of Workers √ó Days = Work (constant)
3. Pressure √ó Volume = Constant (gas)
4. Cost per item √ó Quantity = Budget (fixed)
```

---

## PART 2: DEEP CONCEPT - WORK AS PROPORTION

### Key Insight

**Work is NOT measured in units, but in "fraction of job completed"**

```
If worker completes job in 6 days:
Worker's rate = 1/6 job per day

In 2 days: completes 2 √ó (1/6) = 2/6 = 1/3 of job
In 6 days: completes 6 √ó (1/6) = 1 whole job (100%)
In 3 days: completes 3 √ó (1/6) = 3/6 = 1/2 of job

Think of it like this:
Job = 100 units
Worker's rate = 100/6 = 16.67 units per day
In 3 days: 3 √ó 16.67 = 50 units (half the job)
```

## PART 3: BASIC TIME/WORK FORMULAS

### Formula 1: Single Worker

```
If worker completes job in d days:
Rate = 1/d job per day
Work in t days = t √ó (1/d) = t/d of job
```

### Formula 2: Multiple Workers (Combined Rate)

```
If A's rate = 1/a and B's rate = 1/b:
Combined rate = 1/a + 1/b

Time for both together:
t = 1 / (combined rate)
```
